Metodologia

Sprints de 2 semanas com pequenas entregas incrementais. Modelo híbrido (ágil + cascata): as etapas seguem uma sequência lógica, mas dentro de cada etapa há flexibilidade para adaptar o escopo conforme o que for descoberto durante a execução.
